Sincronización en tiempo real · Firebase · v29 Optimizado
1
Crea tu proyecto Firebase (gratis, 5 minutos)
1. Ve a console.firebase.google.com → Crear proyecto → ponle un nombre (ej: produtrack-planta)
2. En el proyecto: menú lateral → Build → Realtime Database → Create database → elige la región → selecciona "Start in test mode"
3. Luego ve a Project Settings (ícono ⚙️) → General → sección "Your apps" → agrega app web (</>) → copia los datos de configuración
2
Pega tu configuración de Firebase aquí
Los datos están en el objeto firebaseConfig que te da Firebase al registrar la app web.
💡 La configuración se guarda en este navegador. Solo necesitas hacerlo una vez por dispositivo. El plan gratuito de Firebase (Spark) permite 1 GB de almacenamiento y 10 GB de transferencia/mes — más que suficiente para producción.
