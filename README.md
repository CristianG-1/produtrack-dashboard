# ProduTrack — Sistema de Indicadores de Producción

Dashboard desarrollado en HTML y JavaScript para el seguimiento 
en tiempo real de órdenes de trabajo en procesos agroindustriales.

## ¿Qué hace?
- Notificación y seguimiento de órdenes de producción
- Indicadores de cumplimiento por línea y máquina
- Visualización en tiempo real para planta y supervisores
- Exportación de reportes

## Tecnologías usadas
- HTML / CSS / JavaScript
- Firebase Realtime Database (versión demo sin conexión)
